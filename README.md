# downloadlr-srv
### youtube-dl
##### Usage
> youtube-dl "url" \-\-output "`desiredname`.%(ext)s"

##### Example
> youtube-dl "https://www.youtube.com/watch?v=Gh6JkwZ-5QM" \-\-output "Wings.%(ext)s"

## Docker usage (Mac and Linux only)
Prerequisites:
- Docker
- Kitematic (optional, GUI for Docker containers management)
> chmod +x deploy.sh
> ./deploy.sh
